# Glide Keys 

Glide Keys is a real-time multiplayer typing battle game built with the MERN stack and Socket.io. Compete against others in real-time to test your typing speed and accuracy!

## Features

- **Real-time Multiplayer:** Race against other players in real-time typing battles using WebSockets.
- **Matchmaking Lobby:** Find opponents quickly and seamlessly.
- **User Authentication:** Secure signup and login using JWT.
- **Personal Profiles & Stats:** Track your performance, typing speed (WPM), and match history.
- **Interactive Arena:** Dynamic typing interface with real-time progress of you and your opponent.

## Tech Stack

**Frontend:**
- React (with Vite)
- Tailwind CSS
- React Router
- Socket.io-client
- Axios

**Backend:**
- Node.js & Express
- MongoDB (with Mongoose)
- Socket.io
- JSON Web Tokens (JWT) for authentication
- Bcrypt for password hashing


## How to Play
1. Create an account or log in.
2. Click on "Find Match" from the home screen to enter the matchmaking lobby.
3. Once an opponent is found, the race begins!
4. Type the provided text as fast and accurately as possible.
5. Check your stats on the Results page after the match.
